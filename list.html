html

Hypertext Markup Language is the standard markup language for documents designed to be displayed in a web browser. It defines the content and structure of web content. It is often assisted by technologies such as Cascading Style Sheets and scripting languages such as JavaScript.

CSS

CSS, or Cascading Style Sheets, is a web design language used to format the layout and style of documents written in a markup language, most commonly HTML. While HTML handles the content and structure of a web page, CSS determines how that content is presented visually. This includes controlling the colors, fonts, spacing, layout, and responsiveness of a web page.
